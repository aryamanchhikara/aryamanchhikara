# Aryaman World

Static multi-page site designed for GitHub Pages.

## Structure

- `index.html` is the homepage with section peeks.
- `pages/` contains dedicated pages for in-progress work, fashion, music, art, and movies.
- `styles/site.css` contains the visual system.
- `scripts/data.js` holds the editable copy and preview data.
- `scripts/main.js` powers the homepage and Three.js background.

## Publish on GitHub Pages

1. Push this folder to a GitHub repository.
2. In GitHub, open `Settings` -> `Pages`.
3. Under `Build and deployment`, choose `Deploy from a branch`.
4. Select your main branch and the `/ (root)` folder.
5. Save, then wait for GitHub Pages to publish the site.

## Update Content

Edit `scripts/data.js` to swap in your real text, artist names, films, designers, and music links.
