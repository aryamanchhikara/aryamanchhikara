# Aryaman Chhikara

Static multi-page site designed for GitHub Pages.

## Structure

- `index.html` is the homepage with section peeks.
- `pages/` contains dedicated pages for in-progress work, fashion, music, art, and movies.
- `styles/site.css` contains the visual system.
- `scripts/data.js` holds the editable copy and preview data.
- `scripts/main.js` powers the homepage and Three.js background.
